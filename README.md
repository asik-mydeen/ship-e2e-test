# ship-e2e-test

End-to-end test app with Supabase integration

## Development

```bash
npm install && npm run dev
```

## Deployment

Every push to `main` auto-deploys to **https://ship-e2e-test.asikmydeen.com** via Dokploy.

## Infrastructure

- **Hosting:** Dokploy on TrueNAS → https://projects.asikmydeen.com
- **Database:** Supabase → https://supabase.asikmydeen.com
- **Domain:** https://ship-e2e-test.asikmydeen.com
